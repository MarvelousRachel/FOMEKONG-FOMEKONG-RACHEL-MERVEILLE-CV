Underwater SLAM System Development

Guangdong Ocean University, 2024

Designed and implemented a complete underwater SLAM solution optimized for turbid and challenging marine environments. The system integrates advanced deep learning techniques with traditional SLAM approaches to provide robust navigation and mapping capabilities for underwater vehicles.

Technologies Used

  • Python
  • PyTorch
  • OpenCV
  • Open3D
  • g2o
  • SuperGlue/SuperPoint

Key Features

  • Custom SLAM pipeline from feature extraction to 3D mapping
  • Adaptive image enhancement for turbid underwater conditions
  • Multi-stage feature matching with fallback mechanisms
  • Water current compensation algorithms in pose estimation
  • g2o-based pose graph optimization with robust kernels
  • Adaptive voxel downsampling for efficient 3D reconstruction
  • Efficient loop closure detection using appearance and geometry
  • Comprehensive visualization tools for trajectory analysis

Robot Car with Sensor Fusion SLAM

Guangdong Ocean University, 2022-Present

Developed a ground-based robot car platform for implementing and testing SLAM algorithms in real-world environments. This project focuses on integrating multiple sensors (stereo camera, LiDAR) with ROS to create a robust mapping and localization system.

Technologies Used

  • ROS/ROS2
  • C++
  • Python
  • LiDAR
  • Stereo Camera
  • Gazebo
  • RViz

Key Features

  • Full ROS integration with custom message types
  • Real-time sensor fusion of camera and LiDAR data
  • Differential drive kinematics implementation
  • Gazebo simulation environment for testing
  • Complete URDF model with accurate physics
  • RViz visualization for real-time mapping
